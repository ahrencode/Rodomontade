ABOUT

Rodomontade is a very simple template for creating a "personal" (or "vanity") site. A sort of retro about.me for masochists :-).

INSTALLATION/CONFIGURATION

To use this template, you need:

  1. A site with support for PHP 5
  2. Copy all files in this distro to your site
  3. Configure some simple values in the file config.php
  4. Create sections (slides) describing your life and activities in the directory sections under your installation
  5. OPTIONAL: 1 Add any CSS customisations in custom.css
  6. OPTIONAL: Add any additional HTML/JS elements (will likely need to be absolute positioned) in footer.html
