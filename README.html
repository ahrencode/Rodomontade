INSTALLATION/CONFIGURATION

To use this "personal site" template, you need to:
  1. A site with support for PHP 5
  2. Configure some simple values in the file config.php
  3. Create sections (slides) describing your activities in the directory "sections" under your installation
  4. OPTIONAL: Add any CSS customisations in custom.css
  5. OPTIONAL: Add any additional HTML/JS elements (will likely need to be absolute positioned) in footer.html
